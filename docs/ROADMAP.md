# Product Roadmap - PsicoEval

---

## 🎯 Phase 1: MVP (Q2 2026)

**Goal**: Validate core value proposition with early adopters

### Core Features
- [x] Project setup and architecture
- [ ] Authentication (professional login/register)
- [ ] Patient management (create, list, edit)
- [ ] Assessment link generation (5 instruments)
- [ ] Patient questionnaire interface
- [ ] Automatic scoring engine
- [ ] Email notifications
- [ ] Results dashboard

### Instruments
- [ ] PHQ-9 (Depression)
- [ ] GAD-7 (Anxiety)
- [ ] PCL-5 (PTSD)
- [ ] AUDIT (Alcohol Use)
- [ ] MMSE (Cognitive)

### Launch Criteria
- All 5 instruments fully functional
- Mobile-optimized patient interface
- Critical item detection working
- Email notifications reliable
- Security audit passed

### Post-MVP (Phase 2-3)
- 500 active professionals
- 10,000 assessments/month

### Scale (Phase 4)
- 5,000 active professionals

---

## 🎨 Design Principles

All phases guided by:
1. **Simplicity First** - Easy for professionals, effortless for patients
2. **Clinical Validity** - Only validated, evidence-based instruments
3. **Privacy by Design** - HIPAA compliance from day one
4. **Mobile-First** - Optimize for patient experience on phones
5. **Accessibility** - WCAG 2.1 AA compliance minimum

---

## ⚠️ Not Planned

Features explicitly out of scope:
- Diagnostic AI (only use validated instruments)
- Direct patient messaging/chat
- Prescribing or treatment delivery
- Social features or community forums
- Insurance billing automation (Phase 4 only)
